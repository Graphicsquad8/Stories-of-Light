📌 Content Active/Inactive Control System
🔹 Feature Overview

Within the website, there are multiple content sections such as:

All Articles
Motivational Stories
Duas
Books
(and any new sections created by Contributors in the future)

👉 Each content item in every section must have an Active / Inactive control system.

🔹 Required Functionality

Super Owner / Owner / Admin:

Can set any content in any section as Active (visible)
Or set it as Inactive (hidden)

📌 Meaning:
Whether a content item is displayed on the website or not —
👉 will be fully controlled from the Admin Panel.

🔹 Behavior Rules

✔ If Active:

The content will be visible on the website

❌ If Inactive:

The content will not be displayed on the website (hidden)
However, it will not be deleted from the database
🔹 Scope

This system will apply to:

All existing sections
Any new sections created in the future

📌 Ad Controls – Page-Level & Content-Level Ad Management
🔹 Page-Level Controls (Settings → Ad Controls)

Inside the Ad Controls option in the Settings section, the following rules apply:

📌 If any Page-level control is turned OFF:

No ads will be shown on that specific page

📌 If turned ON:

Ads will display normally on that page

⚠️ Important:

This control applies only to page-level ads
Ads inside content (content-level ads) will remain unchanged
